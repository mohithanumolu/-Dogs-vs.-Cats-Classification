# Dogs vs Cats Classification  

## Overview  
This project classifies images of dogs and cats using a **Convolutional Neural Network (CNN)**. It is trained on the Kaggle Dogs vs Cats dataset and achieves high accuracy in binary image classification.  

## Dataset  
- Source: [Kaggle Dogs vs Cats](https://www.kaggle.com/c/dogs-vs-cats/data)  
- Classes: Dog, Cat  
- Preprocessing: Image resizing, normalization, train/test split  

## Methodology  
- **Model Architecture**: CNN with convolutional, pooling, and dense layers  
- **Training**: Cross-entropy loss, Adam optimizer  
- **Evaluation**: Accuracy, confusion matrix, classification report
